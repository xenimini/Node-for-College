# 📝 NoteBook - Менеджер заметок с авторизацией

Простое веб-приложение для создания и управления заметками. Каждый пользователь видит только свои заметки.

## ✨ Возможности

- Регистрация и вход в систему
- Создание, просмотр, редактирование и удаление заметок
- Заметки привязаны к конкретному пользователю
- Автоматическая дата создания/обновления заметки
- Свёртка длинных заметок (кнопка "Ещё...")
- Красивый адаптивный интерфейс

## 🛠️ Технологии

- Node.js
- Express.js
- HTML/CSS (адаптивный дизайн)
- JavaScript (нативном, без фреймворков)

## 🚀 Запуск проекта

```bash
npm install express
node server.js
```

Сервер запустится на `http://localhost:3000`

## 📁 Структура проекта

```
├── server.js              # Главный сервер
├── utils/
│   ├── userManager.js     # Работа с пользователями
│   └── fileManager.js     # Работа с заметками
├── public/
│   ├── index.html         # Главная страница
│   ├── login.html         # Страница входа
│   └── app.js             # Клиентская логика
└── data/                  # Данные (создаются автоматически)
    ├── users.json
    └── notes.json
```

## 📂 Структура данных

**users.json**
```json
{
  "id": 1,
  "username": "user",
  "password": "123"
}
```

**notes.json**
```json
{
  "id": 1,
  "owner_id": 1,
  "title": "Заголовок",
  "content": "Текст заметки",
  "date": "01.01.2025, 12:00:00"
}
```

## 🔗 API

| Метод | Маршрут | Описание |
|-------|---------|----------|
| POST | `/api/register` | Регистрация |
| POST | `/api/login` | Вход |
| GET | `/api/notes` | Получить все заметки пользователя |
| POST | `/api/notes` | Создать заметку |
| PUT | `/api/notes/:id` | Обновить заметку |
| DELETE | `/api/notes/:id` | Удалить заметку |
| GET | `/api/stats` | Получить статистику |
