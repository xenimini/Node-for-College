# 🍊 NoteBook

Заметки, которые не увидят чужие глаза (если ты не покажешь им свой пароль).

- Регистрация и вход по логину/паролю
- Заметки привязаны к пользователю
- Длинные заметки можно свернуть/развернуть кнопкой «Ещё...»
- Никаких БД — всё хранится в JSON-файлах
- Работает на чистом Node.js (без сторонних зависимостей)

> Сделано для души и колледжа 🧡

## 🚀 Запуск

```bash
# Клонируй репозиторий
git clone https://github.com/твой-логин/notebook.git

# Перейди в папку проекта
cd notebook

# Запусти сервер (без установки зависимостей — чистый Node.js)
node server.js
Открой браузер и перейди по адресу: http://localhost:3000

Первый запуск? Создай аккаунт через «Зарегистрироваться» и начинай писать заметки 📝

🔐 Особенности
Пароли пока без хэширования (учебный проект)

Заметки разных пользователей не пересекаются

ID заметок переиндексируются после удаления
